# Smart Procurement UI

This `frontend` folder contains a React + Tailwind CSS + TypeScript design system for the Smart Procurement platform.

## Setup

1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm run dev`

## What’s included

- `src/components/layout` — reusable dashboard layout, sidebar, and topbar
- `src/components/ui` — buttons, cards, inputs, status pills
- `src/pages` — page templates for dashboard, procurements, bidding, suppliers, analytics, reports, notifications, users, profile, settings, and auth
- `tailwind.config.js` — design tokens, dark mode ready
- `vite.config.ts` — fast development with Vite

## Notes

This UI is built as a standalone frontend application. To integrate with the existing PHP backend, connect React pages to backend APIs or use the existing backend as a data source.
